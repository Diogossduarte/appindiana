# Painel do Representante

Painel de estoque e vendas para representante farmacêutico.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (pode ser público ou privado)
2. Faça upload de todos esses arquivos para a branch `main`
3. Vá em **Settings → Pages → Source → GitHub Actions**
4. Faça um push qualquer — o workflow publica automaticamente

O site abrirá em: `https://seu-usuario.github.io/nome-do-repositorio/`

> Não precisa de Node.js, npm ou build — o `index.html` funciona direto.
